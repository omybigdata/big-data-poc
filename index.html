Welcome to OMyBigData GitHub Pages.

OMyBigData researches and develops tool and software to solve critical big data problems that may not be solved by main stream big data vendors. Our goal is to make people comfortable with big data technology by adding critically need features to existing big data to solutions.

Here is the list of things we are working on right now:

  • Central Key Management System
  • Data Encryption in Hadoop
  • Secondary & Spatial Index for hadoop
